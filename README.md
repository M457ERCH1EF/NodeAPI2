# NodeAPI2
A Node app that accepts 2 arguments:
1. portnumber - port for node to run
2. message

To run node on port 8080:
```
> node index.js p:8080 m:hello
```

## To install
* clone this repo
```
> git clone https://github.com/M457ERCH1EF/NodeAPI2.git

> cd NodeAPI2
```
* install the app
```
> npm install

```
## To run the app
```
> node index.js p:8080 m:hello
```
